# CustomKali

## Using the Script

Donwload the file and execute as admin user, no **root**!

```bash
wget https://raw.githubusercontent.com/0x4r2/CustomKali/main/customizeme.sh

chmod +x customizeme.sh

./customizeme.sh
```
